# Branding — ARVI Logistics

## Company name in the UI

Use **ARVI Logistics** everywhere users see the company name (header, page titles, headings, image `alt` text, metadata).

Do **not** use “S & S Brokerage”, “S&S Brokerage”, or “S & S Brokerage Inc.” in new copy.

## Where it is defined today

- **`app/layout.tsx`** — `<title>` and site `metadata`
- **`app/components/Header.tsx`** — logo wordmark (mark letter is **A**) and tagline
- **`app/components/CarriersPartnerSection.tsx`** — partner headline and truck image `alt`

## Not automatically renamed (update when you have final assets)

- **Contact email** in `Header.tsx` (`mailto:` and visible address) may still use the previous domain until a new inbox exists.

## npm package name

`package.json` `name` is **`arvi-logistics-site`** for this repo.
